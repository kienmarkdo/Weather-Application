# Weather Application

First web development project using basic HTML, SCSS and JavaScript. Uses the Open Weather Map API to retrieve weather data and displays a mobile-first responsive design. Features accessibility considerations, the geolocation API, persistent storage, Fetch API with async/await.

## Overview

### Description

The page is split into four sections.

- Top search bar.
- Main forecast display.
- Navigation buttons (from left to right).
  - Current Location forecast (Longitude and Latitude)
  - Home forecast
  - Save current location as Home
  - Toggle unit (Celcius/Farenheit)
  - Refresh forecast
- 6-day forecast.
